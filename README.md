# eDonate
## Using Quasar Framework and Express JS to build a Mobile App and Web App.

![Images](front-end/src/assets/preview.png "Preview")

## Project setup
```
npm install
```

### Run Back-End Services
```
npm run start
```

### Run Front-End Services
```
quasar dev
```
### Run on Android Emulator
```
quasar dev -m cordova -T android
```

### Compiles and minifies for production
```
quasar build
```
